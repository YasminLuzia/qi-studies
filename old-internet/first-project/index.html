6 Tags que foram descontinuadas do HTML5

  • <u> = Sublinhado - Esta tag foi substituída por <ins>.

    Exemplo: <ins>Palavra</ins>

  • <acronym> = Definir siglas - É mais utilizada atualmente a tag <abbr>.

    Exemplo: <abbr title="Hyper Text Markup Language">HTML</abbr>

  • <dir> = Definir uma lista - Caiu em desuso por conta do <ul>.
    Exemplo: <ul>
    • <li>Item da lista</li>
      • </ul>
  • <big> = Letras maiúsculas - Foi inutilizada pois é configurado pelo CSS como 'text-tranform:uppercase;'.
    Exemplo: No CSS: .maiusculo{
    text-tranform:uppercase;
    }
    E na pagina é utilizada como:

    <p class="maiusculo">MAIÚSCULO</p>

  • <center> = Centralizar texto - Não é mais utilizada pois é configurado pelo CSS como 'text-align:center;'.
    Exemplo: No CSS: .centralizar{
    text-align:center;
    }
    E na página é utilizada como:

    <p class="centralizar">Centro</p>

  • <strike> = Riscar a palavra - Essa tag parou de ser utilizada e foi substituida por <del> ou <s>.
    Exemplo: <del>Riscado</del>
    ou
    <s>Riscado</s>
