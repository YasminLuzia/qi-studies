Bootstrap - Framework Front-end

O bootstrap é um framework profissional, voltado para o público desenvolvedor front-end. O mesmo é dotado de diversos recursos de estilização CSS, efeitos JS, além de jQuery. Para utilizar o bootstrap é necessario vincular ao arquivo ao arquivo bootstrap.min.css e bootstrap.min.js ou utilizar por CDN - links.

Exemplos:

<button type="button"> Botão </button>

Exemplo de Botão de Bootstrap

<button type="button" class="btn btn-default"> Botão </button>
