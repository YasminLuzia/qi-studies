Semântica do HTML5

A semântica no HTML5 serve para descrever seu significado ou função, tanto para o navegador, quanto para o desenvolvedor.
Os elementos semânticos no HTML5 são elementos com um significado pré-definido.

<header>

O elemento <header> é um container de conteúdo introdutório ou um conjunto de links para navegação.

<nav>

O elemento <nav> define um conjunto de links para navegação.

<section>

O elemento <section> define uma seção de um documento.

<aside>

O elemento <aside> é um conteúdo a parte do conteúdo principal.

<article>

O elemento <article> especifica um conteudo auto-explicativo, independente.

<footer>

O elemento <footer> é o rodapé de uma seção ou documento.
