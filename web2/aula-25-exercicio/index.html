Laços de repetição e Arrays

Laços While

O propósito da declaração while é simples. Ele dirá ao PHP para executar as declarações aninhadas repetidamente, enquanto a expressão do while forem avaliadas como TRUE.
Exemplo:

<?php
$i = 1;
while ($i <= 10) {
 echo $i++;
}
?>

Laços Do While

Os laços do-while é muito similar aos laços while, com exceção que a expressão de avaliação é verificada ao final de cada iteração em vez de no começo.
Exemplo:

<?php
$i = 0;
do {
 echo $i;
} while ($i > 0);
?>

Laços For

A primeira expressão do for é avaliada (executada), uma vez, incondicionalmente, no início do laço.
No começo de cada iteração a segunda expressão é avaliada. Se a avaliada como TRUE, o laço continuará e as instruções aninhadas serão executadas. Se avaliada como FALSE, a execução do laço terminará.
No final de cada iteração, a terceira expressão é avaliada (executada).
Exemplo:

<?php
for ($i = 1; $i <= 10; $i++) {
 echo $i;
}
?>

Arrays

Um array no PHP é na verdade um mapa ordenado. Um mapa é um tipo que relaciona valores a chaves. Este tipo é otimizado para várias usos diferentes: ele pode ser tratado como um array, uma lista (vetor), hashtable (que é uma implementação de mapa), dicionário, coleção, pilha, fila e provavelmente mais. Assim como existe a possibilidade dos valores do array serem outros arrays, árvores e arrays multidimensionais.
Exemplos:

<?php
$array = array("Hello, ", "world!");
var_dump($array);
?>

Laços For Each

O construtor foreach fornece uma maneira fácil de iterar sobre arrays. O foreach funciona somente em arrays e objetos, e emitirá um erro ao tentar usá-lo em uma variável com um tipo de dado diferente ou em uma variável não inicializada. Ela possui duas sintaxes:
A primeira forma, itera sobre arrays informados na array_expression. A cada iteração, o valor do elemento atual é atribuído a $value e o ponteiro interno do array avança uma posição (então, na próxima iteração, se estará olhando para o próximo elemento).
A segunda forma var, adicionalmente, atribuir a chave do elemento corrente a variável $key a cada iteração.
Exemplos:

<?php
$array = array(1, 2, 3, 4);
foreach ($array as &$value) {
 $value = $value * 2;
} // $array agora é array(2, 4, 6, 8)
unset($value); // quebra a referência com o último elemento
?>
 <?php
$array = array(1, 2, 3, 4);
foreach ($array as $key => $value) {
// $array[3] será atualizado com cada valor de $array...
 echo "{$key} => {$value} ";
 print_r($array);
} ?>
